# Video Upload Progress

Мини-проект на JavaScript с использованием axios и AbortController. 
Позволяет загружать видео с отображением прогресса и возможностью отмены запроса.

## Как запустить
1. Открой index.html в браузере.
2. Выбери видеофайл.
3. Нажми "Загрузить".
4. Можно отменить загрузку кнопкой "Отменить".
